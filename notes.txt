-100 to 100 for pwm is the duty cycle range (80 means 80 percent duty cycle)
On many MCUs, including PIC32, Output Compare (OC) is the peripheral that compares:
    the current timer count against a programmed compare value and then changes the pin when the compare match happens.
PWM mode is a specific output compare mode where the output behavior is defined over the whole timer period.

    For example, in PWM mode 1:

        output is active while CNT < CCR4
        output becomes inactive when CNT >= CCR4
        counter resets at ARR

    So:

        ARR sets the PWM period
        CCR4 sets the duty cycle

In PWM mode 1 with active-high polarity, the output is high while: CNT<CCR4
